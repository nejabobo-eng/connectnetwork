# ConnectNetwork Website (V1)

Static, visual-first site built with Next.js, TypeScript and Tailwind CSS.

## Tech
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide Icons
- Deployed on Vercel (recommended)

## Scripts
- npm run dev — start local dev
- npm run build — build for production
- npm run start — start production server

## Contact Form
Replace the action URL in app/contact/page.tsx with your Formspree form id.

## Notes
- No backend, auth or payments.
- Images use Unsplash placeholders. Replace with your own assets.
